# Flexbox

* Nos permite posicionar os elementos dentro da caixa.
* Controle em uma dimensão (horizontal ou vertical)
* Alinhamento, direcionamento, ordenar e tamanhos

```css
div.parent {
	display: flex;
}
```

## Flex-direction
* Qual a direção do flex: horizontal ou vertical
* row | column

## Alinhamento
* justify-content
* align-items
