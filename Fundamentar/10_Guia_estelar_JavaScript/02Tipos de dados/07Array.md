# Array 
O tipo de dados Array, ou vetor, também é estrutural, e consiste em uma lista de dados agrupados, um Array é denotado com os caracteres [ ] , por exemplo: [ "Leite", "Ovos", "Manteiga", 1, 2 ] .

    * Array (Vetores)
        * Uma lista
        * Agrupamento de dado

        ["Mayk", 36]]

## Exemplos
```js
    console.log([
        "Leite",
        "Ovos",
        2,
        3,
        3 + 4
    ])
```
